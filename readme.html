dev2dev Mashup Demo

This demo shows how a developer can build a very simple Mashup using the technologies described in the tutorial. It presumes that two services have already been built:

  • A REST service that returns a list of dev2dev Tech Days venues with addresses
  • A service that converts an address into a Google Maps image

Installing the Demo

All that is required is a web server. Copy the contents of the zip file onto the web server.

Generating a Google Maps API Key

You will need to generate a Google Maps API key (it's free) and update the key in ajaxGoogleMapsMashup.html. To generate a key, you will be asked to provide a URL. Your URL will look something like: http://ajaxmashup.googlepages.com. Navigate here to generate your key.

You will need to insert your key in the script URL for Google Maps: 

<script src="http://maps.google.com/maps?file=api&v=2&key=YOUR_KEY_HERE"
  type="text/javascript"></script>

Running the Demo

The finished product can be accessed at http://yourhost/yourpath/ajaxGoogleMapsMashup.html.
